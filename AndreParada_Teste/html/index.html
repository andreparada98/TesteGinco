Teste de formulário

Desafio proposto pela Ginco como teste de estágiarios.

Crie um dataset simples, com poucos registros, e desenvolva um formulário utilizando HTML e JavaScript. O formulário deve conter os campos:

  1. nome do cliente,
  2. CPF
  3. data de nascimento

O cliente deve incluir seu nome no campo nome do cliente e os demais dados devem ser preenchidos automaticamente. Aproveite para estudar o método que o FLUIG já disponibiliza para isso, conforme a documentação: https://style.fluig.com/javascript.html#fluig-autocomplete.

-Modus Operandi-

  1. Fazer o HTML Básico
  2. Fazer as formatações básicas
  3. Definir os atributos
  4. Fazer o dataset no JS
  5. Fazer o autocomplete
  6. Brincar com o CSS
